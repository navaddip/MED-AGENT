# MED-AGENT# 🩺 MedAgent

An AI-powered Clinical Decision Support System for Chest X-ray Analysis using the NIH ChestX-ray14 Dataset.

## Features

- Exploratory Data Analysis
- Data Cleaning
- Multi-label Classification
- Patient-wise Data Splitting
- Custom PyTorch Dataset
- Data Augmentation
- Weighted Sampling
- Deep Learning Pipeline

## Dataset

NIH ChestX-ray14 Dataset

112,120 Chest X-ray Images

14 Disease Classes

## Tech Stack

- Python
- PyTorch
- Torchvision
- OpenCV
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
